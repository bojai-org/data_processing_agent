# Data Processing Agent

This repository contains the internal experiments and prototyping code used to develop the BojAI data processing agent. Please check our [website](https://bojai-documentation.web.app/how-to-guides/how-to-use-the-agent/) for details about the agent. 

## Features

* Processes your data from natural language explanation
* Simple interface for Bojai-integrated use
* Uses Ollama models

## Part of the Bojai Ecosystem

This agent is one of the building blocks in **Bojai** — an open-source framework for streamlined ML pipelines.
Learn more: [https://bojai.org](https://bojai.org)

## Repository Purpose

This repo includes both the production agent code and early experiments that shaped its design.
Feel free to explore, modify, or repurpose components for your own workflows.
